PHI-base data visualization tool

PHI-base is a database containing genetic information about pathogen-host interactions. This page provides two tools to visualize parts of this dataset. The stacked bar chart shows which genes are relevant for a set of diseases, and the force-directed graph shows what phenotypical effects effects are caused by these diseases.

Usage:

  • Select one or more diseases from the 'All diseases' drop-down menu
  • Click 'Show stacked bar chart' for the gene distribution across diseases
  • Click 'Show force-directed graph' for the mutant phenotype distribution across diseases
